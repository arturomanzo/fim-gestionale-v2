Migrazione Firebase → Supabase

Trasferisci tutti i dati del gestionale v1 nel nuovo database PostgreSQL

1 Connessione Supabase

Inserisci le credenziali del tuo progetto Supabase (le trovi in Project Settings > API)

2 Verifica Schema Database

Controlla che le tabelle siano state create eseguendo 001_schema_completo.sql nel SQL Editor di Supabase

3 Importa Dati Firebase

Esporta il database Firebase (Console > Realtime Database > ... > Esporta JSON) e caricalo qui

📁
Trascina il file JSON qui
oppure clicca per selezionare

4 Anteprima Dati

Riepilogo dei dati trovati nel file Firebase

5 Esegui Migrazione

Trasferisci i dati nel database Supabase. Il processo e' sicuro: usa UPSERT (inserisce o aggiorna, non duplica).
